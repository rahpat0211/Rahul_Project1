Branches

It is a branch that Github displays when anyone visits the repository. A Default Branch is used for pull requests and code commits.

Merge

It is used to combine two branches after finding a common base commit.

Commit

Snapshots of your repository at specific times. Which results in a story of your history of the repository.

Tag

Used to mark a point in the repository history, usually a version number for a release.

Repository

It is a collection of files of different versions of a project.
