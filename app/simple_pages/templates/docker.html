Container

A unit of software that packages up code and all dependencies so the application runs quickly and reliably.

Images

It is an image that includes everything needed to run an application.



Commands I Learned

Command Explanation
docker run -itp ‘Port #’ ‘Repository name: tag name’
docker run -p ‘Port #’ ‘Repository name: tag name’
docker run -dp ‘Port #’ ‘Repository name: tag name’		 Creates a container from a given image and starts the container.
docker stop ‘Container ID’ To stop a running container
docker ps Lists all the containers that are up and running
docker images List of all the docker images
cd Change Directory
pwd What your current path is
